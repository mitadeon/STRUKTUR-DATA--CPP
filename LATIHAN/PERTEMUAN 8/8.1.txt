Algoritma Dasar Double Ended Queue (Deque)
a. Inisialisasi

Tujuan: Menyiapkan Deque agar siap digunakan.
Langkah-langkah:

Mulai
Tentukan ukuran maksimum deque, misalnya MAX.
Set variabel Front = -1 dan Rear = -1 untuk menandakan deque masih kosong.
Siapkan array Deque[MAX] untuk menampung data.
Selesai

b. Insert Sebuah Record dari Kanan

Tujuan: Menambahkan elemen baru di ujung kanan deque (rear end).
Langkah-langkah:

Mulai
Periksa apakah deque penuh, yaitu ketika (Front == 0 && Rear == MAX - 1) atau (Front == Rear + 1)
Jika ya, tampilkan pesan “Deque penuh”.
Berhenti.
Jika deque kosong (Front == -1), maka set Front = Rear = 0.
Jika Rear == MAX - 1, maka atur Rear = 0 (bergerak melingkar).
Jika tidak, tambahkan Rear = Rear + 1.
Simpan data baru ke Deque[Rear].
Selesai.

c. Insert Sebuah Record dari Kiri

Tujuan: Menambahkan elemen baru di ujung kiri deque (front end).
Langkah-langkah:

Mulai
Periksa apakah deque penuh, yaitu ketika (Front == 0 && Rear == MAX - 1) atau (Front == Rear + 1).
Jika ya, tampilkan pesan “Deque penuh”.
Berhenti.
Jika deque kosong (Front == -1), maka set Front = Rear = 0.
Jika Front == 0, maka atur Front = MAX - 1 (bergerak melingkar).
Jika tidak, kurangi Front = Front - 1.
Simpan data baru ke Deque[Front].
Selesai.

d.Delete Sebuah Record dari Kanan

Tujuan: Menghapus elemen dari ujung kanan deque (rear end).
Langkah-langkah:

Mulai
Periksa apakah deque kosong, yaitu Front == -1.
Jika ya, tampilkan pesan “Deque kosong”.
Berhenti.
Ambil data dari Deque[Rear].
Jika Front == Rear, berarti hanya ada satu elemen, maka set Front = Rear = -1.
Jika Rear == 0, maka set Rear = MAX - 1.
Jika tidak, kurangi Rear = Rear - 1.
Selesai.

e.Delete Sebuah Record dari Kiri

Tujuan: Menghapus elemen dari ujung kiri deque (front end).
Langkah-langkah:

Mulai
Periksa apakah deque kosong, yaitu Front == -1.
Jika ya, tampilkan pesan “Deque kosong”.
Berhenti.
Ambil data dari Deque[Front].
Jika Front == Rear, berarti hanya satu elemen, maka set Front = Rear = -1.
Jika Front == MAX - 1, maka set Front = 0.
Jika tidak, tambah Front = Front + 1.
Selesai.
